# feature-demo/AGENTS.md

Instructions for the demo feature module.

## Module Purpose

`feature-demo` demonstrates how consumers should use `ftt-ui-lib`.

## Responsibilities

- Showcase FTT UI components.
- Provide sample screens and sample state.
- Demonstrate light/dark mode where useful.
- Demonstrate typical usage patterns for consumers.

## Forbidden

- Do not add reusable UI components here.
- Do not publish code from this module as library API.
- Do not add real business logic.
- Do not depend on private app-only assumptions.

## Rules

- Use `ftt-ui-lib` as a consumer would.
- Keep sample data local and obvious.
- Use clear demo names.
- Demo screens should be easy to copy into documentation.
